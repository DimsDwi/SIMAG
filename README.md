## 📖 Deskripsi Sistem
SIMAG (Sistem Informasi Magang) merupakan sebuah platform perangkat lunak berbasis web dengan arsitektur Client-Server yang dirancang untuk mendigitalisasi dan mengintegrasikan seluruh proses administrasi program magang secara terpusat. Sistem ini bertujuan untuk menyederhanakan alur birokrasi dan memfasilitasi komunikasi antara mahasiswa, dosen pembimbing, serta instansi perusahaan melalui fitur-fitur terpadu seperti pendaftaran, pengelolaan dokumen persuratan, pengisian logbook harian, hingga validasi dan penilaian akhir. Secara teknis, antarmuka pengguna (Front-End) dirancang agar responsif dan intuitif, sementara bagian Back-End difokuskan pada manajemen database yang kokoh dan keamanan autentikasi untuk memastikan seluruh riwayat magang, status persetujuan, dan nilai terekam dengan akurat.
