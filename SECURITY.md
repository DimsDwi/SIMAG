# Security Policy

## Supported Versions

We currently support the following versions of SIMAG:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within SIMAG, please send an e-mail to the maintainers privately rather than opening a public issue.

We will review all security vulnerabilities and will try to patch them in a timely manner. We appreciate your efforts to responsibly disclose your findings.
